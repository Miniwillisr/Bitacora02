Display

Ideas Principales

Es una de las propiedades mac cmomplejas de CSS. Esta propiedad establece el tipo de caja que genera cada elemento

Cada elemento tiene un valor de visualizacion predeterminado ya sea Block o Inline para la mayoria

Apuntes

Tipos de Display

  • Block

    • Siempre comienza con una nueva linea y ocupa todo el ancho posible o disponible. (div, h1, p, form, header)

  • Inline

    • No Comienza en una nueva linea y solo ocupa el ancho que sea necesario. (a, span, img)

  • Inline-block

    • Permite establecer ancho y una altura en el elemento, se respetan los marhin y padding que con Inline no son soportados.

  • None

    • Se usa comunmente con JavaScript para ocultar y mostrar elementos cuando el usuario realiza alguna accion como pausar un boton o enlaces. (Hace que el elemento no genere ninguna caja)

  • table

    • Permite Asignar un comportamiento de tabla a los elementos que haya dentro. Se pude usar otra seria de display que correspondan con los roles de distintas etiquetas HTML

Resumen

La propiedad display a pesar de ser algo compleja y con muchas variantes puede llegar a ser una de las mejores opciones para mostrar elementos en html gracias a su versatibilidad
