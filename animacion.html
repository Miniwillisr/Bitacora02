Animación

Ideas Principales

CSS permite animar la mayoria de los elementos HTML sin usar JavaScript

Puede haber varios estados de animacion hasta un maximo de 100
(0%{}, 50%{}, 100%{})

Para que una animacion funcione, se debe vincular la Animacion a un elemento

Apuntes

Sintaxis

@keyframes nombre{
from{background-color:red;
}
to {background-color:yellow;
width:500px; }

Se tiene que especificar la duracion con animation-duration ya que el valor el predeterminado es 0 pero si se mantiene en este valor la animacion no se apreciara

  • iteration-count

    • Especifica el numero de veces que se reproduce la animacion

  • Delay

    • Especifica un retraso en la Animacion

  • Timing-function

    • Especifica la curva de velocidad

Resumen

La animaciones son muy dinamicas ya que nos pueden ayudar a crear efectos visuales combinando varios colores o desplazando ciertos elementos de un sitio a otro
