Position

Ideas Principales

La propiedad Posistion consta de 5 atributos:

  • Static
  • Relative
  • Fixed
  • Absolute
  • Sticky

Apuntes

  • Static

    • Los elementos HTML se posisionan estaticos de manera predeterminada. Estos elementos no se ven afectados por Top, left, right o bottom.

  • Relative

    • Este elemento es desplazado respecto a su posicion Original. El desplazamiento de controla con las Propiedades Top, Bottom, left y Right.

  • Fixed

    • Se ubica con relacion a la ventana grafica, lo que significa que siempre esta fijo. Un elemento fijo no deja un espacio donde normalmente se ubicaria

  • Absolute

    • Su posicion se indica mediante Top, Bottom, left y Right. dependen del elemento contenedor cuando hay una posicion diferente a Static

  • Sticky

    • Se desplaza en duncion del usuario (Scroll) Alterna entre Relativo y Fijo, dependiendo del desplazamiento. Se posiciona de forma relativa hasta que encuentra una posicion de desplazamiento dada en la ventana grafica, y luego se pega como fixed

Resumen

La porpiedades de Position son muy imortantes, ya que nos ayudan a poder tener un orden o colocar ciertas elementes de nuestro html como queramos. Si sabemos utilizar correctamente esta propiedad podremos crear muchas variedades
